# 第五版 - 搜索与索引基础设施 PRD
## 1. 功能定位
第五版要求搜索从“第四版骨架”进入“真实可重建、可验证、可部署”的状态。

## 2. 当前版本状态
状态：已完成

## 3. 本轮必须完成
1. 服务端真实执行搜索重建
2. 搜索文档写入 `search_documents`
3. 重建记录写入 `search_rebuild_runs`
4. Redis 写入搜索运行态状态
5. `/admin/search` 展示真实 MySQL / Redis 状态和重建结果

## 4. 暂不要求的内容
1. 向量检索
2. 推荐系统
3. 分布式搜索集群
4. 独立异步索引 worker
