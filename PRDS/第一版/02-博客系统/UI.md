# 博客系统 UI

## 当前状态

状态：已完成第一版正式实现，可继续打磨

## 当前 UI 方向

博客系统保持“可公开阅读”的质量标准，而不是只做一个能跑的内容壳子。

当前页面已经形成以下视觉特征：

- 列表页有精选首条与常规列表的节奏差
- 详情页以长文阅读舒适度优先
- 标签、时间和摘要已经纳入统一层级
- 代码块和引用块已有正式样式

## 与当前代码一致的结构

### 列表页

- 页面引导区
- 标签预览侧栏
- 精选文章卡片
- 次级文章卡片
- 常规文章列表

### 详情页

- 页面引导区
- 返回列表入口
- 标题、时间、标签
- Markdown 正文排版
