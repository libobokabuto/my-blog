# 博客系统 PRD

## 功能定位

博客系统是第一版最核心的正式内容系统，负责承载长期发布的文章内容和站点最主要的阅读体验。

## 第一版状态

状态：已完成第一版正式实现

说明：

- 博客列表页 `/blog` 已完成正式实现
- 博客详情页 `/blog/:slug` 已完成正式实现
- 基于 Markdown 的博客读取、排序、标签展示已接通
- 上下篇导航与第一版基础站点输出能力已完成

## 第一版当前实现范围

博客系统当前已经包含：

1. 博客列表页
2. 博客详情页
3. 发布时间展示
4. 标签展示
5. 按时间排序
6. Markdown 正文渲染

## 第一版已补齐的内容

1. 上一篇 / 下一篇文章导航
2. 更完整的站点级输出
3. 与 RSS / sitemap 的衔接

## 第一版暂不要求

- 评论系统
- 站内搜索
- 数据库驱动筛选
- 后台编辑
- 复杂目录侧栏
