# 笔记系统开发计划

## 当前状态

状态：已完成第一版基础实现

原因：

- 博客主链路已经可运行
- `notes` 已完成第一版基础接入

## 开发目标

在现有 `Leptos SSR + Markdown` 架构下，为 `notes` 接入正式页面与内容读取能力。

## 建议实施顺序

1. 明确 `notes` front matter 字段
2. 准备 `content/notes` 示例内容
3. 实现 `/notes` 路由
4. 实现笔记列表组件结构
5. 接入排序与摘要展示
6. 暂不实现单篇详情页

## 完成标准

- `/notes` 页面可访问
- 能从 `content/notes` 读取内容
- 页面风格与博客同属一个站点，但信息密度更轻
