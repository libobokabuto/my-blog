# 首页 UI

## 当前状态

状态：已完成第一版首轮实现，可继续打磨

## 当前 UI 方向

首页维持“编辑感内容站”而不是“产品落地页”或“在线简历”的方向。

当前页面已经形成以下视觉特征：

- 暖色背景与纸感层次
- 衬线标题搭配简洁正文
- 首屏双栏布局
- 内容卡片与说明卡片交错排布

## 现状与代码一致的要点

- 首屏左侧是身份说明与 CTA
- 右侧是“当前主题”和“第一阶段目标”卡片
- 下方是最新博客、最近笔记、代表项目三个入口面板
- 再往下是“第一版重点”和“当前边界”说明区

## 后续优化方向

- 首页中的 `notes` 和 `projects` 入口区已经使用真实内容
- 继续增强卡片层级和移动端节奏
- 在不破坏克制感的前提下提高内容区可扫描性
