# 个人主页 PRD

## 功能定位

这个模块用于在第三版中新增一个更完整的个人主页或公开工作台页面。

它不是简单替代 `about`，而是：

- 作为站点入口枢纽
- 连接内容、项目、学习主题和最近动态
- 展示适量的个人统计与公开状态

## 当前状态

状态：未开始

## 第三版目标

- 新增一个正式的个人主页页面
- 明确主页与 `about` 页分工
- 让主页承担“介绍 + 导航 + 最近动态 + 轻量统计”的职责

## 当前实现范围

建议优先实现：

1. 当前状态 / Now
2. 最近博客、最近笔记、最近项目
3. 当前重点主题入口
4. 轻量统计模块

## 建议增强

在主体完成后，再考虑：

1. GitHub 提交热力图
2. 技术栈或语言统计
3. 站点运行状态
4. 服务器资源概览

## 暂不要求

- 复杂实时监控大屏
- 登录后个人面板
- 数据库驱动统计系统
