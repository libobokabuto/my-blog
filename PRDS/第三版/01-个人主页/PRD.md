# 个人主页 PRD

## 功能定位

这个模块用于在第三版中新增一个更完整的个人主页或公开工作台页面。

它不是简单替代 `about`，而是：

- 作为站点入口枢纽
- 连接内容、项目、学习主题和最近动态
- 展示适量的个人统计与公开状态

## 当前状态

状态：已在第一批正式实现中完成

## 第三版目标

- 新增一个正式的个人主页页面
- 明确以 `/me` 作为第三版个人主页正式入口
- 明确主页与 `about` 页分工
- 让主页承担“介绍 + 导航 + 最近动态 + 轻量统计”的职责

## 当前实现范围

建议优先实现：

1. 当前状态 / Now
2. 内容入口，连接首页、`about`、`blog`、`notes`、`projects`
3. 最近动态，优先聚合最近博客与最近笔记，并展示当前重点项目
4. 当前重点主题入口
5. 轻量统计模块

## 本轮补充约束

- 不新增数据库，不做持久化统计
- 最近动态与统计基于现有 Markdown 内容聚合
- 允许新增服务端主页聚合视图模型，为第四版后端接入预留结构

## 当前实现结果

- `/me` 已正式接入主导航
- 已完成与首页、`about`、`blog`、`notes`、`projects` 的导航关系打通
- 已完成最近动态、内容入口、重点项目、关注主题与轻量统计
- 统计仍是内容驱动的轻量统计，没有进入持久化或后台化实现

## 建议增强

在主体完成后，再考虑：

1. GitHub 提交热力图
2. 技术栈或语言统计
3. 站点运行状态
4. 服务器资源概览

## 暂不要求

- 复杂实时监控大屏
- 登录后个人面板
- 数据库驱动统计系统
