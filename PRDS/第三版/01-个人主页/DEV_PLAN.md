# 个人主页开发计划

状态：已在第一批正式实现中完成

## 开发目标

为第三版新增一个个人主页，使它成为站点公开导航枢纽。

## 本轮目标

1. 明确主页内容结构
2. 确定与 `about` 的分工
3. 先做静态与内容驱动部分
4. 预留后续统计模块位置
5. 同步首页与主导航，让 `/me` 成为第三版正式入口之一

## 建议实施顺序

1. 先同步第三版总文档与模块文档状态
2. 设计 `/me` 页面结构与首页、`about` 的职责边界
3. 接入最近内容、重点项目与内容入口
4. 接入轻量统计与最近动态聚合
5. 调整主导航与跨页跳转关系
6. 视需要接入 GitHub / 状态类增强模块

1. 已完成第三版总文档与个人主页模块文档同步
