# 个人主页 UI

## 当前状态

状态：未开始

## 视觉方向

个人主页应更像“公开工作台”而不是“监控面板”。

应保持：

- 内容站气质
- 编辑感与长期项目感
- 清晰分区和较强的个人辨识度

## 页面气质

主页建议同时具备：

- 个人介绍页的温度
- 内容导航页的清晰度
- 项目现场的进行中感觉

## 建议模块层次

1. 顶部：身份与当前状态
2. 中部：最近动态与重点项目
3. 下部：主题入口与轻量统计
4. 附加区：GitHub 热力图、技术统计、运行状态

## 后续优化方向

- 强化主页与其他内容页的视觉区分
- 让统计模块更克制，不抢正文入口
- 让页面既能浏览，也能快速跳转到内容
