# 搜索升级开发计划

## 当前状态

状态：已在第一批正式实现中完成

## 开发目标

把当前搜索从“能搜到”推进到“更容易找到真正相关内容”。

## 本轮目标

1. 调整搜索结果排序
2. 增加类型筛选和标签筛选
3. 优化结果摘要和命中提示

## 建议实施顺序

1. 先同步文档状态
2. 扩展搜索结果数据结构
3. 改造搜索参数与服务端排序
4. 接入搜索页筛选 UI

## 完成标准

- 结果排序不再只按类型粗排
- 用户可按类型和标签收窄结果
- 每条结果都有更清楚的命中解释

## 本轮完成情况

1. 已完成搜索结果结构扩展
2. 已完成服务端搜索排序与筛选改造
3. 已完成搜索页筛选 UI 与结果表达升级
