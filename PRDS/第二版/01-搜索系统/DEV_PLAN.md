# 搜索系统开发计划

## 当前状态

状态：已完成

## 开发目标

实现一个基于当前 Markdown 内容源的可用搜索页 `/search`。

## 本轮目标

- 建立覆盖三类内容的基础搜索索引
- 实现 `/search` 路由与 query 参数读取
- 补齐搜索输入、结果列表和命中提示

## 建议实施顺序

1. 明确搜索索引字段
2. 建立简单内容索引
3. 实现 `/search` 路由
4. 实现搜索输入与结果列表
5. 补结果高亮或命中提示

## 本轮完成标准

- `/search` 可访问
- 可检索 `blog`、`notes`、`projects`
- 搜索结果有类型、摘要与命中提示
- 空结果与空查询状态清晰

- 已实现 `/search`
- 已建立覆盖三类内容的基础搜索索引
- 已补齐输入区、结果列表、高亮、空态和命中提示
