# 第二版项目展示 PRD

## 功能定位

第二版项目展示的目标，是把第一版项目列表升级成完整展示链路。

## 当前状态

状态：已完成

## 第二版目标

- 实现 `/projects/:slug`
- 展示项目背景、目标、状态、技术栈与外链
- 让项目页更像作品档案而不是简单卡片集合

## 当前实现范围

- 保留项目列表页
- 新增项目详情页 `/projects/:slug`
- 详情页展示标题、状态、技术栈、外链与 markdown 正文
- 列表页中的项目卡片需可直接进入详情页
- 列表页会补充状态分组或状态感知
- 首页中的代表项目入口可直接进入详情页

## 暂不要求

- 自动同步 Git 仓库数据
- 后台管理
- 数据库
