# 标签与归档开发计划

## 当前状态

状态：已完成

## 开发目标

为站点建立最基础但正式可用的标签归档系统。

## 本轮目标

- 统一 `blog` 与 `notes` 的标签读取和归档数据结构
- 实现 `/tags/:tag` 路由与标签结果页
- 补齐从列表页、详情页进入标签归档的入口

## 建议实施顺序

1. 统一标签数据提取方式
2. 实现 `/tags/:tag` 路由
3. 组织归档列表页结构
4. 让列表页与详情页增加标签跳转入口

## 本轮完成标准

- `/tags/:tag` 可访问
- 标签页能同时展示对应博客与笔记
- 标签跳转入口覆盖博客与笔记主要页面

## 完成结果

- 已实现 `/tags/:tag`
- 已补齐博客列表、博客详情、笔记详情的标签跳转入口
- 已实现标签页中博客与笔记的分区展示

## 完成标准

- 标签归档页可访问
- 归档页能展示对应内容
- 标签跳转路径自然
